# 21DAY — Курс по ИИ для помогающих специалистов

Лендинг обучающего курса «21 день с ИИ» для психологов, тарологов, нумерологов и других помогающих специалистов.

## Локальная разработка

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve

# PHP
php -S localhost:8080
```

После этого: http://localhost:8080

## Деплой

Статический сайт. Можно разместить на:

- **GitHub Pages** — включите в настройках репозитория
- **Netlify / Vercel** — перетащите папку или подключите git
- **Любой хостинг** — загрузите файлы через FTP/SFTP

После деплоя обновите в `index.html`:
- `og:url` — полный URL страницы
- `og:image` — путь к изображению для соцсетей (рекомендуется 1200×630)

## Робокасса

В `script.js` в функции `handleSubmit` — заглушка для подключения оплаты. При интеграции:

1. Отправьте данные формы на ваш бэкенд
2. Бэкенд создаёт заказ и возвращает URL Робокассы
3. Редирект: `window.location.href = robokassaUrl`
